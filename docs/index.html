app-preflight

Catch App Store & Google Play rejections before you submit — a fast, zero-config CLI for iOS and Android.

GitHub All checks Download

Why

Most App Store “red” rejections are for boring, mechanical, detectable reasons — an empty permission string, a missing privacy manifest, a debuggable build, a missing demo account. preflight encodes those reasons as checks so a rejection shows up in a pull request instead of three days into review.

Quick start

cargo install --git https://github.com/ubadeaslan/app-preflight preflight-cli

preflight check .                 # scan a project (iOS and/or Android)
preflight check ./MyApp.ipa       # or a compiled artifact
preflight check . --format sarif  # SARIF for GitHub code scanning

What it does

Source scan

Info.plist, entitlements, PrivacyInfo.xcprivacy, AndroidManifest, Gradle, network security config.

Store metadata

App Store Connect & Google Play APIs: demo account, privacy/support URLs, screenshots, descriptions.

Compiled binary

Unzips IPA/APK: Mach-O (UIWebView, private frameworks, IDFA/ATT) and DEX (hidden APIs, secrets, 64-bit).

CI-native

SARIF code scanning, Markdown job summaries, a baseline to fail only on new issues, and a GitHub Action.
